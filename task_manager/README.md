# Task Manager
A simple web application for managing tasks, built with Flask and SQLite.
## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed or pending
- Search tasks by title
## Technologies
- Python
- Flask
- Flask-SQLAlchemy
- Flask-WTF
- SQLite
- HTML
- CSS
- JavaScript
## Installation and Run

1. Activate virtual environment
2. Install dependencies:

pip install -r requirements.txt

3. Run the project:

python run.py

4. Open in browser:

http://127.0.0.1:5000
